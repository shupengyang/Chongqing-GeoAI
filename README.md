# 🏙️ Chongqing-GeoAI
**重庆主城区 GeoAI 空间态势感知与智能交互平台**

## 📖 项目简介
本项目以被称为“8D魔幻城市”的重庆为主体，针对其复杂的山地地形与立体的三维交通网络，构建一个集空间态势感知与智能交互于一体的 GeoAI 平台。

通过整合网络数据爬取、GIS 空间分析与数据可视化技术，本项目旨在为山地城市的规划与交通治理提供可视化的数据决策支持。

## ✨ 核心功能 (持续开发中)
- [ ] **多源数据采集:** 通过高德地图 API 爬取重庆主城区的 POI 与路网数据。
- [ ] **三维拓扑建模:** 结合 DEM 高程数据，重点攻克跨江大桥与穿山隧道的复杂立体网络拓扑构建。
- [ ] **空间数据清洗:** 对海量的多源地理空间数据进行清洗、转换与标准化处理。
- [ ] **智能交互模块:** 开发前端交互界面，实现交通拥堵态势与空间资源分配的可视化呈现。

## 🛠️ 技术栈
* **数据获取:** Python (Requests, BeautifulSoup), 高德地图开放平台 API
* **空间分析:** ArcGIS Pro, QGIS
* **数据处理:** Pandas, GeoPandas
* **数据可视化:** 待定 (例如 Kepler.gl, Mapbox, 或 ECharts)

## 🗺️ 开发计划
- **第一阶段:** 编写 Python 爬虫脚本，定向获取重庆主城核心区 (如渝中区、江北区等) 的基础地理与 POI 数据。
- **第二阶段:** 处理高程数据，在 GIS 软件中完成三维路网拓扑关系的构建与空间分析。
- **第三阶段:** 开发并部署交互式可视化分析面板。
